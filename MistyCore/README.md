## Features
- Custom hoe dirt textures
- Custom location backgrounds, just like Island North's sky.
- Adding critters to locations depending on chance and/or conditions.
- Requirements to fish in a location, and alternative fishes to give if those requirements aren't met.
- "Object hunt" command, which lets you make searches similar to the egg hunt.
- Custom tile actions.
- Custom game state queries.
- Custom trigger actions.
- Custom event commands.

This mod makes use of ContentPatcher to be edited.
